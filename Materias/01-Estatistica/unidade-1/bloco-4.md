
## Representação gráfica de séries estatísticas
Gráficos servem para mostrar dados de forma visual, rápida e fácil de entender.

Em vez de ler números em uma tabela, você enxerga o comportamento nos dados.

Todo gráfico deve ter:

- Título: fica na parte superior, explica o que o gráfico mostra.
- Fonte: quando necessário - fica na parte inferior, indica de onde vieram os dados.

Regra geral
O gráfico deve ser:
- fácil de entender
- claro
- direto

Se for confuso, está errado

---

## Tipos de gráficos
Quando usar cada um

### Gráfico de barras
- Barras na horizontal
- Usado para comparar categorias

Exemplo:
- população por região
- vendas por produto

### Gráfico de colunas
- Barras na vertical
- Muito parecido com o de barras

Usado para:
- comparações também
- evolução simples

### Gráfico de setores ou pizza
- Mostra partes de um todo

Exemplo:
- porcentagem de mercado
- divisão de gastos

### Histograma
- Parecido com barras
- usado para dados contínuos (intervalos)

Exemplo:
- distribuição de salários
- alturas

### Gráfico de disperção
- Mostra relação entre duas variáveis

Exemplo:
- altura vs peso
- idade vs salário

### Polígono de frequência
- linha ligando pontos de frequência
- usado para ver tendência / distribuição

Não existe “um gráfico melhor”, existe o gráfico mais adequado para o tipo de dado.

Dica:
- Comparar categorias → barras ou colunas
- Mostrar porcentagem → setores (pizza)
- Dados contínuos → histograma
- Relação entre variáveis → dispersão

---

## Gráfico de disperção
Um dos mais importantes

Mostra a relação entre duas variáveis

Tem dois eixos:
- eixo horizontal (x)
- eixo vertical (y)

Cada ponto representa um par de valores

Exemplo do slide:
- PIB per capita (eixo x)
- Expectativa de vida (eixo y)

Cada ponto = um país

relação entre duas granzas -> dispersão

## Gráfico de barras


Comparar categorias

Ele tem:
- barras horizontais
- eixo vertical -> categorias (variável qualitativa)
- eixo horizontal -> valores (variável quantitativa)

Exemplo: região vs. número de pessoas

## Gráfico de colunas
É praticamente o mesmo que o de barras, mas invertido.

- Categorias → eixo horizontal - variável qualitativa 
- Valores → eixo vertical - variável quantitativa

Se tem:
- Barras = horizontal
- Colunas = vertical

## Gráfico de setores (pizza)
Mostrar partes de um todo

Características:
- formato circular
- dividido em "fatias"
- geralmente em porcentagem

Exemplo:
- % de alfabetizados vs não alfabetizados

interessante usar % na maioria das vezes

muitas fatias com valores próximos → difícil de interpretar

Dica:
- poucas categorias → ok
- muitas categorias → ruim

## Histograma
Muito importante - LEMBRAR QUE É INTERVALOS. 

Valor de X a Y

Mostra distribuição de dados contínuos

Características
- Parece gráfico de colunas
- mas:
	- não tem espaço entre barras
	- trabalha com intervalos (classes)

Eixos:
- horizontal -> intervalos
- vertical -> frequência

Se tiver:
- intervalos
- frequência

→ é histograma

## Polígono de frequência
Mostra tendência de distribuição

Liga os pontos do topo das barras do histograma

Ele serve para:
- visualizar o comportamento dos dados
- ver crescimento ou queda

## Resumo geral
- Dispersão → relação entre variáveis
- Barras → comparação (horizontal)
- Colunas → comparação (vertical)
- Setores → partes de um todo
- Histograma → distribuição (intervalos)
- Polígono → tendência da distribuição

## Pegadinhas clássicas

1. Dispersão ≠ comparação  
    → é relação
2. Histograma ≠ barras comuns  
    → usa intervalos
3. Pizza com muitos dados  
    → ruim (difícil de interpretar)

Um estudo estatístico completo de uma população deve se basear no método estatístico, que é composto por uma série de fases:
- Definição do problema.
- Delimitação do problema.
- Planejamento para a obtenção dos dados.
- Coleta dos dados.
- Apuração dos dados.
- Apresentação dos dados.
- Análise dos dados.
- Interpretação dos dados.

## Ponto médio (Pm)
Quando você trabalha com dados agrupados em intervalos, vocẽ não sabe o valor exato

Então você usa o ponto médio da classe

Formula:

Ponto médio = (Limite inferior + Limite superior) / 2

Você pega o intervalo e calcula o valor “do meio”.

Vamos supor que precisamos fazer uma análise ou media do seguinte intervalo:

- Salários de 1000 a 2000

Não temos como saber o valor único, então assumimos que todos aqueles valores serão representados pelo valor do meio, que é o Pm.

Nesse caso, faríamos 1000 + 2000 / 3 = 1500

É como se disséssemos:

“Já que não sei os valores exatos, vou assumir que, em média, eles estão por volta de 1500”
