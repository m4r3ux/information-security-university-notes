## Media ponderada
A média ponderada é usada quando nem todos os valores têm a mesma importância.

Ou seja
- alguns "pesam mais" que outros

## Fórmula

Média ponderada = (soma de valor x peso) / (soma dos pesos)

Soma de valor x peso:

8x1 = 8

7x1 = 7

4x3 = 12

Somar os valores:

8 + 7 + 12 = 27

Somar os pesos:

1 + 1 + 3 = 5 

27 / 5 = 5,4

Resumindo: 

Para cada valor, multiplique ele pelo seu peso, some tudo isso depois divida pela soma de todos os pesos.

## Mediana
Mediana = valor que fica exatamente no meio dos dados ordenados

### Passo a passo lógico 

1- Você tem um conjunto de dados

Exemplo: 
`2,03 | 2,41 | 1,99 | 1,82 | 2,06 | 2,03 | 2,01`

2- Ordenar (colocar em ordem crescente)
`1,82 | 1,99 | 2,01 | 2,03 | 2,03 | 2,06 | 2,41`

Organize do maior pro menor para enxergar o meio.

3- Encontrar o meio
Agora entra a lógica principal:

- Total de valores: 7 (impar)
- O valor do meio é o 4o 

Resultado:
`Mediana = 2,03`

### Regra geral

Caso 1: quantidade IMPAR
Fórmula da posição:

`posição = (N + 1) / 2`

Exemplo, o último número é 7, então:
Médiana = (7 + 1) / 2 = 4

Caso 2: quantidade PAR
Exemplo: 
1, 2, 3, 4

Dois valores do meio: 2 e 3
`Mediana = (2 + 3) = 2,5`

Pra achar o meio do N que é par, podemos pegar as posições com:
```
posição 1 = N / 2
posição 2 = (N / 2) + 1
```
